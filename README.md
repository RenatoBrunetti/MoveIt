## Getting Started

It's very simple. Trust me.

### Prerequisites

- Node.js
- Yarn

### Installation

1. Clone the repo

```sh
git clone https://github.com/RenatoBrunetti/MoveIt.git
```

2. Install Packages

```sh
yarn
```

## Build and Start

Build and start the application

```sh
yarn build
yarn start
```
